The CONSOLE screen

What is console screen? Console screen is where you can see information on what activities you are doing or to show commands etc. It usually shows activities. How to open console screen (PC / Laptop) You can put this command : (Ctrl + Shift + I) OR Right click and select inspect. You will be able to see various tabs at the top. For most users the second tab is console. How to print on console screen You need to go to the file Java Script (JS) and add : console.log ("Put text here")
